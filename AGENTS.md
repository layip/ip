# Agent Instructions

Use these repository-level instructions when working in this repo.

## Skills

Imported skills are documented in `.chatgpt/skills/SKILLS.md`.

When a task involves documents, PDFs, slides, or spreadsheets, follow the matching skill workflow before editing or generating artifacts.

## General workflow

- Inspect existing files before changing them.
- Keep changes small and focused.
- Prefer clear, readable code and documentation.
- Run relevant checks or tests when available.
- Summarize what changed and mention any checks that could not be run.
